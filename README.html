Installation and Getting Started guides for MiramoPDF and MiramoEnterprise

This repository contains the DITA maps for the following Miramo guides:

  • "Miramo_DITA-OT_GettingStarted_2.0.ditamap" - The MiramoPDF DITA-OT plugin Getting Started Guide
  • "MiramoEnterprise_install_guide_for_dita.ditamap" - Covers installing MiramoEnterprise, MiramoPDF and the DITA-OT plugin on all platforms
  • "MiramoPDF_install_guide_for_dita.ditamap" - The MiramoPDF installation guide, covers installing desktop versions of MiramoPDF on Windows and macOS, as well as trial versions of MiramoPDF on Linux, and installing free MiramoPDF DITA-OT plugin.
